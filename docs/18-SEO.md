# 6. Technical SEO Requirements

## Performance
ClearMargin must remain fast.

Targets:
- First Contentful Paint < 2s
- Minimal JavaScript
- No heavy animations
- Optimized images

Minimal design = SEO advantage.

---

## Mobile First
Google indexes mobile versions first.

Requirements:
- readable font size
- no horizontal scrolling
- fast mobile load

---

## Semantic HTML
Use real HTML meaning:

<header>
<main>
<section>
<article>
<footer>

Avoid div-only layouts.

---

## Accessibility (Indirect SEO)
Required:
- alt text for images
- proper heading order
- readable contrast

Accessible sites rank better.
